## Installation

```bash
$ npm ci
```

Add `.env` file in the root folder of the repository with following content.

`DATABASE_MONGODB_URI=mongodb://localhost:27017`

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Check the app

The App should be available on `localhost:3001/api`
